# 🎓 Face Recognition Attendance System
A smart, seamless, and secure attendance tracking system built using **Flask, OpenCV, Face Recognition (dlib), and SQLite**.
This system automatically detects and recognizes faces in real-time and marks attendance without manual intervention.

---

## 🚀 Features

- 👤 Student Registration with Face Capture
- 🎥 Real-Time Face Recognition via Webcam
- ✅ Automatic Attendance Marking
- ❌ Prevents Duplicate Attendance (Same Day)
- 📊 Admin Dashboard with Attendance Statistics
- 📁 CSV Report Export
- 🔎 Student Management (Search & Delete)
- 💾 SQLite Database Integration
- 🎨 Clean and Responsive UI

---

## 🛠️ Tech Stack

- **Backend:** Flask (Python)
- **Face Recognition:** OpenCV + face_recognition (dlib)
- **Database:** SQLite
- **Frontend:** HTML, CSS
- **Data Handling:** NumPy
